# Frontend Mentor - Social Media Dashboard

Solucao do desafio **Social media dashboard with theme switcher** do Frontend Mentor.

O projeto esta focado apenas na interface da dashboard, com dados mockados em
`src/data/dashboard.ts`. Nao ha login, banco de dados, APIs externas ou integracao
com redes sociais reais.

## Funcionalidades

- Layout responsivo para mobile e desktop.
- Cards de seguidores e estatisticas diarias com dados mockados.
- Alternancia entre tema claro e escuro.
- Estados de hover nos cards e no controle de tema.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
